• 1.Virtual Dom

    <=>Virtual dom is the light weight copy of rela dom.
    <=>Virtual dom have two copies one Virtual is present in the mounting phase and another vurtual dom contains updated state values.
    <=>This two virtual doms will compare each other this method is called "DIFFING ALOGORITHM".
    <=>After this Updated will be done in Real Dom this method is called "RECOINCILIATION".

  • 2.React Features

    <=>React uses the virtual dom instead of real dom.
    <=>It follows the Uni-directional data flow.
    <=>It uses the component based structure for reusability of code.

  • 3.Functional Compo VS Class Compo

    <=>Class Component have the render method to return JSX.
    <=>Functional Component is a plain Java Script function it does not contain render method it returns JSX.
    <=>Class Component uses the State and setState methods to Update the values.
    <=>Functioanl Component uses the hooks concept UseState method to update the Values.
    <=>Class Component uses the LifeCycle methods.
    <=>Functional Component use the UseEffect method it is the alternative of LifeCycle Methods.
