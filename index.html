• 1.Virtual Dom

    <=>Virtual is the copy to the real DOM.
    <=>The react.js contains TWO virtual doms, first virtual dom is get created whenever the component is get mounted on the screen for the first time.
    <=>Another is get created when ever the changes happened in the component by clicking button.Second virtual dom is stored the updated values.
    <=>This two virtual doms will compare each other for the new changes this method is called "DIFFING ALOGORITHM".
    <=>The state change values updated into the Real Dom this method is called "RECOINCILIATION".

  • 2.React Features

    <=>Easy learning curve
    <=>JSX
    <=>We use virtual dom instead of Real Dom in React.js
    <=>React-Native

  • 3.Functional Compo VS Class Compo

    <=>In Class Component we are using render to return the JSX part
    <=>Functional Component does not contain render method to returns JSX.
    <=>Class Component uses the State and setState methods to Update the values.
    <=>Functioanl Component uses the hooks concept UseState method to update the Values.
    <=>Class Component uses the LifeCycle methods for the data manupulations.
    <=>Functional Component use the Hooks method for the data manupulations.
